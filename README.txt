В качестве сборщика проектов мы используем Apache Maven.

Для чистой сборки проекта следует выполнить:

Для запуска приложения необходимо иметь установленый Apache Maven, JDK 6 или выше, и выполнить:
1) в консоли перейти в каталог Workspqce/VRS
2) выполнить команду mvn clean(для WIN систем имеется скрипт clean.bat)
3) mvn package(для WIN систем имеется скрипт package.bat)
4) скрипт launch.bat
  UNIX(в терменале):
	sh target/bin/webapp
5)в браузере переходим на страницу:
	localhost:8080/


Если вы не желаете собирать проект самостоятельно, то достаточно выполнить п. 4-5.

!! Если вам нужен war-файл(деплоится на сервер локальный):
mvn compile war:war
Файл появится в директории target.
